2. OpenCV (Open Source Computer Vision)

A biblioteca líder para processamento de imagens.

import cv2
import matplotlib.pyplot as plt

# Carregar imagem
img = cv2.imread('image.jpg')

Nota: O OpenCV utiliza o padrão BGR por padrão, enquanto a maioria das ferramentas usa RGB.

06

A Questão do BGR vs RGB

Ao exibir uma imagem do OpenCV com o Matplotlib, as cores Red e Blue estarão trocadas.

# Conversão necessária
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
plt.imshow(img_rgb)

07

Escala de Cinza (Grayscale)

Reduz a complexidade da imagem para 1 único canal de intensidade (0-255).

gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
  • Menor custo computacional.
  • Foco em formas e texturas, ignorando a cor.
08

Dimensões e Atributos

# Retorna (Altura, Largura, Canais)
print(img.shape)

# Total de pixels
print(img.size)

As imagens são tratadas como arrays multidimensionais (Tensores) do NumPy.

09

Redimensionamento (Resize)

Ajustar o tamanho da imagem para requisitos de entrada de modelos.

# Largura, Altura
resized = cv2.resize(img, (224, 224))

Atenção: Mudar a proporção (aspect ratio) pode distorcer os objetos na imagem.

10

Recorte (Crop)

Utilizamos slicing de matrizes para selecionar uma Região de Interesse (ROI).

# img[y_start:y_end, x_start:x_end]
roi = img[100:400, 200:500]
11

Normalização de Dados

Escalar os valores de pixels [0, 255] para o intervalo [0, 1].

img_normalized = img / 255.0

Essencial para acelerar a convergência durante o treino de Redes Neurais.

12

Espaço de Cor HSV

Mais robusto que o RGB para variações de iluminação.

  • Hue: Matiz (Cor pura).
  • Saturation: Vivacidade.
  • Value: Brilho.
hsv = cv2.cvtColor(img,
cv2.COLOR_BGR2HSV)
13

Filtro por Cor (Masking)

low = (35, 100, 100) # Ex: Verde baixo
high = (85, 255, 255) # Ex: Verde alto
mask = cv2.inRange(hsv, low, high)

# Aplicação
res = cv2.bitwise_and(img, img, mask=mask)
14

Suavização e Ruído

Gaussian Blur

Suaviza bordas e remove ruído de alta frequência.

cv2.GaussianBlur(img, (5,5), 0)

Median Blur

Ótimo para remover ruído do tipo "sal e pimenta".

cv2.medianBlur(img, 5)
15

Detecção de Bordas: Canny

Identifica mudanças bruscas de intensidade.

# Threshold 1 e 2
edges = cv2.Canny(gray, 100, 200)

Fundamental para contagem de objetos e OCR.

16

Limiarização (Thresholding)

Transforma a imagem em binária (Preto ou Branco).

ret, thresh = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

Se o valor do pixel > 127, torna-se 255. Caso contrário, 0.

17

Morfologia Matemática

Erosão

Diminui o objeto (remove detalhes pequenos).

Dilatação

Aumenta o objeto (fecha buracos pequenos).

kernel = np.ones((5,5), np.uint8)
erosion = cv2.erode(mask, kernel, iterations=1)

3. Blocos Básicos de uma CNN

Diferente de ML tradicional, CNNs aprendem a extrair características automaticamente.

Convolução

Extrai características (bordas, texturas) usando filtros/kernels.

Pooling

Reduz a resolução espacial (reduz parâmetros) mantendo o mais importante.

Flatten

Transforma a matriz em um vetor para a camada densa.

19

O que é Pooling?

Operação de subamostragem. O mais comum é o Max Pooling.

[Image representing a 2x2 Max Pooling operation]

  • Reduz dimensões: 28x28 → 14x14.
  • Torna a rede invariante a pequenas translações.
20

4. Transfer Learning

Utilizar uma rede pré-treinada em um grande dataset (ex: ImageNet) para uma tarefa específica.

Vantagens

  • Menos dados necessários.
  • Menor tempo de treino.
  • Melhor performance inicial.

Aplicações

VGG, ResNet, MobileNet e a família YOLO.

21

Métricas: Classificação

  • Acurácia: Percentagem de acertos totais.
  • Precisão: De todos os positivos previstos, quantos são reais?
  • Recall: De todos os positivos reais, quantos foram encontrados?
  • F1-Score: Média harmónica entre Precisão e Recall.
22

Métricas: Detecção

IoU (Intersection over Union): Mede a sobreposição entre o desenho real e o previsto.

mAP (mean Average Precision): Média da precisão em diferentes níveis de confiança.

23

5. YOLOv8 (Ultralytics)

YOLO: You Only Look Once. Detecção de objetos em tempo real.

from ultralytics import YOLO
model = YOLO('yolov8n.pt')
results = model('image.jpg')

Modelos: n (nano), s (small), m (medium), l (large), x (extra large).

24

Treino Customizado (YOLOv8)

# Requer ficheiro data.yaml (config do dataset)
model.train(data='data.yaml', epochs=100, imgsz=640)

O Roboflow é a ferramenta recomendada para organizar e exportar estes datasets no formato YOLO.
